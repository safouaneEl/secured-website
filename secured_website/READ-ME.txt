
Étapes pour exécuter le projet
1)Configurer les informations de la base de données :
    Accédez au fichier includes/informations.php.
    Modifiez les variables $host, $user, $password, et $db avec les informations correctes pour se connecter à votre base de données.

2)Installer et configurer XAMPP (vous pouvez travailler avec votre control panel) :

    Si XAMPP n'est pas déjà installé, téléchargez-le et installez-le à partir du site officiel.
    Ouvrez le panneau de contrôle XAMPP (XAMPP Control Panel).
    Démarrez les services Apache et MySQL en cliquant sur les boutons "Start" à côté de chaque service.

3)Placer le projet dans le répertoire approprié :

    Copiez le dossier du projet dans le répertoire xampp/htdocs.
    Assurez-vous que tous les fichiers du projet sont bien placés dans ce répertoire.

4)Accéder au projet :

    Ouvrez votre navigateur web.
    Tapez http://localhost/secured_website  dans la barre d'adresse.
    Le site devrait se charger avec les informations de base de données que vous avez configurées.
